## Algorithms Implemented

1. Selection Sort (with comparisons and swaps)
2. Insertion Sort (with comparisons and swaps)
3. Count Duplicate Elements using Hashing

---

## 1. Selection Sort
Selection sort repeatedly selects the minimum element from the unsorted part and places it at the beginning.

- Time Complexity: O(n²)
- Space Complexity: O(1)

---

## 2. Insertion Sort
Insertion sort works the way we sort playing cards in hand.

- Best Case: O(n)
- Worst Case: O(n²)
- Space Complexity: O(1)

---

## 3. Count Duplicate Elements
This program counts the total number of duplicate elements using hashing.

- Time Complexity: O(n)
- Space Complexity: O(n)
